一、数组的三种定义方式

  1. 定义一个空数组

    var arr = [ ];初始化一个空长度的数组

    var arr = new Array(); 创建内存

  2. 定义了一个有值的数组

    var arr2 = [1,3,4,5,6];

    bar str2 = ["good","hello","ok"];

  3. 先定义后赋值的数组

    var arr3 = [ ];

    arr3[0] = "thank";

    arr3[1] = "welcome";

      slice
      • 知识概括
      • 代码展示

      • slice是指定在一个数组中的元素创建一个新的数组，即原数组不会变

        Slice是从已有数组中截取值,将截取的值赋给新的数组（新数组为空，有值的无效），且元数组的值不变。

        slice(start,end);start:下标 end截取的个数(或者是start+1)

      • var color = new Array('red','blue','yellow','black');

        var color2 = color.slice(1,3);

        alert(color);

        alert(color2);

        red,blue,yellow,black
        blue,yellow;
      splice
      • 知识概括
      • 代码展示

      • splice是JS中数组功能最强大的方法，它能够实现对数组元素的删除、插入、替换操作，返回值为被操作的值

        splice方法--添加、删除，修改元素

        语法： splice(index,count,items（值）);

        如果count等于0或者是负数的话。那么就是在下标index前面追加元素

        如果count大于0的话,那么久是从当前元素开始截取count位，把items放入到截取的位置.

        如果count大于当前数组长度的话，那么他就是截取到当前数组长度位

        index是必填项，count可以不填写,且代码一定要像后面这么写

        如果count不定义或者定义了一个大于当前数组的长度的数字的时候，那么它会从当前下标开始截图到结尾

      • var str7=[12,13,14,15,16,17,18,19];

        str7.splice(2,5);

        alert("变化后："+str7.toString());12,13,19（就是删除了中间的五项）

        str7.splice(3,2,"浙江，江苏");

        alert("变化后："+str7.toString());
        12,13,14,浙江，江苏,17,18,19(就是将中间两项替换成浙江、江苏)

        str7.splice(3,0,"浙江");

        alert("变化后："+str7.toString());
        12,13,14,浙江,15,16,17,18,19(在14后面追击元素)

      三、数组小总结

      1.数组下标都是从0开始，也就是说数组里面的第一个元素的下标是0

      2.获取数组的长度是:length

      3.第一个元素：arr[0] = "xxx"; 最后一个元素:arr[length-1]="xxx"

      4.当一个数组下标越界的时候，返回undefined
