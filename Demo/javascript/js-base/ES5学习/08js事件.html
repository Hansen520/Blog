三、焦点事件(onfocus获取焦点和onblur失去焦点)---引入jQuery说明---

应该是聚焦的意思，当鼠标点击时消失，鼠标点另一个时出现

案例(点击后只见分晓)

显示代码

<input type="text" value="请输入姓名" class="in_text" t_title="请输入姓名"/>

<input type="text" value="破壳日" class="in_text" t_title="破壳日"/>


<script type="text/javascript">

/*focus 该元素获取焦点时*/

$("input.in_text").focus(function(){

if ($(this).val() == $(this).attr("t_title"))

{

$(this).val('');

}

});

/*blur 该元素失去焦点时*/

$("input.in_text").blur(function(){

if ($(this).val() == "")

{

$(this).val($(this).attr("t_title"));

}

});

</script>

四、onchange（change）事件

---onchange 事件常结合对输入字段的验证来使用,点击一次就生效。当用户改变输入字段的内容时，会调用函数,一般用于表单联动---

案例：

代码展示

<select onchange="myChange(this)">

<option value="0">--请选择月--</option>

<option value="1">1月</option>

<option value="2">2月</option>

<option value="3">3月</option>

<option value="4">4月</option>

<option value="5">5月</option>

</select>


<script type="text/javascript">

function myChange(val){

alert(val.value);

}

</script>

五、hover事件---引入jQuery说明---

当鼠标经过时，触发的事件,此事件可以利用js控制也可以利用css样式控制

案例

hover me!
显示代码

<div class="hover">hover me!</div>


<script type="text/javascript">

$(".hover").hover(function(){

$(this).animate({"width":"600px"},1000);

},function(){

$(this).animate({"width":"100px"},1000);

});

</script>
